# Retro Green Snake

A lightweight browser Snake game with an old-school monochrome green terminal aesthetic, built in plain HTML, CSS, and JavaScript.

## Vision
Create a nostalgic arcade experience that is:
- instantly playable in any modern browser,
- dependency-free and easy to understand,
- simple to extend with new modes, mechanics, and visual themes.

## Features
- Fully playable Snake gameplay (movement, food, growth, collisions, game over).
- Monochrome green CRT-inspired UI style.
- Controls: Arrow keys and WASD.
- Start/Restart and Pause/Resume controls.
- Difficulty settings that affect speed and food scoring.
- Selectable grid sizes (16x16, 24x24, 32x32).
- Persistent high score using `localStorage`.
- Retro beep sound effects using Web Audio API.

## Run locally
No installation required.

1. Clone this repository.
2. Open `index.html` in your browser.

## Project structure
- `index.html` — app layout and game controls.
- `styles.css` — retro UI styles.
- `script.js` — game logic and audio.
- `AGENTS.md` — contributor/agent project guidance.
